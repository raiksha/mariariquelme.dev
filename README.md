# Monorepo - mariariquelme.dev

Este repositorio contiene el código fuente de mi ecosistema web personal, organizado por subdominios.

**Estado:** En desarrollo.

## 📁 Estructura del Proyecto

- `apps/landing/`: Sitio principal (mariariquelme.dev) - HTML/CSS/JS.
- `apps/portfolio/`: Mi portafolio profesional - HTML/CSS/JS.
- `apps/blog/`: Artículos, tutoriales y reflexiones - Astro.
- `apps/constanza/`: Perfil interactivo - React.
- `apps/dashboard/`: Proyectos interactivos (app.) - React.

## 🚀 Despliegue

Cada carpeta en `apps/` está configurada para desplegarse de forma independiente en su respectivo subdominio.
